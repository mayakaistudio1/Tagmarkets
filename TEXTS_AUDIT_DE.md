## 10. LanguageContext — ключи, используемые в коде, но НЕ на текущих страницах

> Эти ключи существуют в LanguageContext, но **не отображаются** на активных страницах (могут использоваться в удалённых/неподключённых компонентах).

| Ключ | Текст DE | Примечание |
|------|----------|------------|
| `home.subtitle` | Ihr Zugang zu den Finanzmärkten | Не используется на HomePage |
| `home.ecosystem` | Komplettes Ökosystem | Не используется |
| `home.broker` | Zuverlässiger Broker | Не используется |
| `home.control` | Volle Kontrolle | Не используется |
| `home.education` | Ausbildung | Не используется |
| `home.mariaGreeting` | (дубликат `chat.greeting`) | Не используется |
| `home.mariaOnline` | Maria online | Не используется |
| `home.askMaria` | Maria fragen | Не используется |
| `quick.whatIsJetup` | Was ist JetUP? | Не используется |
| `quick.howToStart` | Wie fange ich an? | Не используется |
| `quick.isSafe` | Ist es sicher? | Не используется |
| `nav.home` | Startseite | Не используется (TabBar hardcoded) |
| `nav.jetup` | JetUP | Не используется |
| `nav.chat` | Chat | Не используется |
| `nav.application` | Anfrage | Не используется |
| `chat.typing` | Maria tippt... | Не используется |
| `video.*` | (все video.* ключи) | Не используются (VideoCallBar имеет свои тексты) |
| `app.*` | (все app.* ключи) | Не используются (нет страницы заявки) |
| `jetup.*` | (все jetup.* ключи) | Не используются (нет страницы экосистемы) |
| `scheduleUhr` | Uhr | Не используется |
